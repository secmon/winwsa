.. _glossary:

Глоссарий
=========

.. glossary::

    Коллектор
      Утилита, управляющая сбором отчета. Коллекторами являются, в основном bat-скрипты или скрипты на других языках (том же AVZ-script), которые содержат в себе функции по сбору отчета с компьютера при помощи провайдеров, архивации и передачи данных. Отчет коллектора может содержать результаты работы нескольких провайдеров.
      
    Отчет
      Совокупность результатов работы коллекторов в т.ч. разобранных. Проще говоря, отчет - набор показателей. Отчет может содержать показатели как по одному компьютеру, так и по нескольким.

    Парсер
      Утилита, осуществляющая преобразование исходного отчета провайдера в нормализованный вид согласно указанной схеме описания показателя и схеме разбора.

    Показатель
      Самостоятельный элемент отчета. Например, элемент отчета, содержащий данные по службе, процессу, заданию планировщика. В контексте IOC-ов - это item.

    Провайдер
      Утилита, осуществляющая непосредственный сбор показателей с компьютеров (Autoruns, AVZ, HiJackThis и т.п.).
