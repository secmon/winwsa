Схема описания показателей
==========================

Схема описания показетелей содержит:

* требования к стуктуре показателя
* перечень общих (единых для всех типов) полей показателя
* классификатор типов
* перечень полей каждого из типов

Схема разбора
=============

Схемы разбора содержат в себе необходимые инструкции по разбору парсером отчета того или иного провайдера.
